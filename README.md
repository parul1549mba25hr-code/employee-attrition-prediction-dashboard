# 📊 Employee Attrition Analysis Dashboard

## Overview

This project presents an interactive **Employee Attrition Analysis Dashboard** built using **Power BI**. It focuses on analyzing employee data to identify patterns, trends, and key factors contributing to attrition.

The dashboard helps organizations make **data-driven HR decisions** to improve employee retention and workforce planning.

---

## Objectives

* Analyze employee attrition trends
* Identify high-risk employees
* Understand key factors influencing attrition
* Provide actionable insights through visualization

---

## Key Features

### Attrition Metrics

* Total Employees
* Attrition Count
* Attrition Rate

### Employee Segmentation

* By Job Role
* By Department
* By Salary/Income Level

### Key Insights

* Impact of overtime on attrition
* Relationship between income and attrition
* High-risk employee categories

### Interactive Dashboard

* Dynamic filters and slicers
* Easy-to-understand visualizations
* User-friendly layout

---

## Tools & Technologies

* **Power BI** – Data visualization & dashboard creation
* **Excel / CSV** – Data source
* **Data Cleaning & Transformation** – Power Query

---

## Dataset

The dataset used in this project contains employee-related information such as:

* Age
* Job Role
* Salary
* Overtime
* Attrition status

*(Note: Dataset is for educational/analytical purposes.)*

---

## How to Use

1. Download the `.pbix` file from this repository
2. Open it in **Power BI Desktop**
3. Explore the dashboard using filters and visuals

---

## Insights & Impact

* Helps HR teams identify **attrition patterns**
* Supports **better decision-making**
